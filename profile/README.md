## 프로젝트 개요

* 프로젝트 기간: 2025.03 ~ 2025.06
* 프로젝트 형태: 경상국립대학교 캡스톤디자인 II (졸업 프로젝트)

### 배경

* 학습 자료에서 제공하는 **연습문제**의 양과 품질이 떨어짐
* 풀이 기록·오답 분석이 흩어져 있어 **학습 진단이 어려움**
* 문제 난이도·유형 다양화가 어려워 **학습 동기 유지가 힘듦**

### 해결 방향

* **LLM 연동(FastAPI)** 으로 텍스트/PDF/오디오 기반 **연습문제 자동 생성**
* **문제집/문제/풀이 로그** 전주기 관리 및 **통계 시각화**
* **폴더링 시스템**으로 개인 학습 리소스 구조화

### 프로젝트 목표

* **학습 효율 향상**: 문제 생성·관리 자동화로 제작/정리 시간 절감
* **사용자 경험 강화**: 문제집·폴더·통계를 한 곳에서 관리
* **학생들의 학습의욕 증진**: 문제집 공유 기능을 통해 경상국립대 학생들이 함께 학습 자료를 활용하고 학습 의욕을 높일 수 있는 환경을 조성

## 주요 기능
1. **회원가입 / 로그인**
   * 이메일 인증 기반 회원가입 및 로그인
   * JWT 기반 인증/인가

2. **문제집 및 문제 관리**
   * 문제집 생성, 조회, 공유
   * 문제 추가, 풀이 기록 저장

3. **AI 문제 생성 (FastAPI 연동)**
   * 텍스트 / PDF 기반 문제 자동 생성
   * 음성 → 텍스트 변환(Whisper) 후 문제 생성 지원

4. **AI 채점 및 키워드 추출**
   * GPT 기반 객관식/주관식 채점
   * 문제별 핵심 키워드 추출 및 오답 키워드 통계 제공

5. **학습 통계**
   * 자주 틀린 키워드 분석
   * 오답률 상위 문제집 보기

6. **폴더링 시스템**

   * 개인 폴더 생성/조회/이동/삭제
   * 과목/단과대별 공유 문제집 폴더 제공
